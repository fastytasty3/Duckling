# Счётчик времени обработки товаров

Программа для учёта времени обработки товаров с помощью сканера штрихкодов. Работает в браузере (Replit) и может быть собрана в установочный `.exe` для Windows.

---

## 1. Как запустить проект в Replit

Проект запускается автоматически. В правой панели будет открыт предпросмотр приложения.

Если нужно запустить вручную:

```bash
# Запустить API-сервер
pnpm --filter @workspace/api-server run dev

# Запустить фронтенд (в отдельном терминале)
pnpm --filter @workspace/scanner-app run dev
```

---

## 2. Как тестировать ввод штрихкода с клавиатуры

1. Откройте главный экран приложения.
2. Выберите оператора и смену в диалоге запуска.
3. Наберите штрихкод на клавиатуре (минимум 4 символа) и нажмите **Enter**.
4. Программа запустит таймер для этого товара.
5. Введите другой штрихкод и нажмите **Enter** — первый таймер остановится, запустится новый.

> **Важно:** поле ввода штрихкода активно всегда, независимо от того, на каком элементе интерфейса находится фокус.

---

## 3. Как подключить USB-сканер

USB-сканер работает как обычная клавиатура — никакой дополнительной настройки не требуется:

1. Подключите сканер к USB-порту.
2. Откройте приложение.
3. Наведите сканер на штрихкод — он автоматически введёт код и нажмёт Enter.

Программа принимает ввод независимо от того, на каком элементе интерфейса сейчас фокус.

---

## 4. Как собрать .exe для Windows

Для сборки установочного файла нужен **Windows** с установленным Node.js 18+.

### Шаг 1: Установить зависимости

```bash
npm install
```

### Шаг 2: Собрать проект

```bash
npm run build   # Сборка фронтенда и бэкенда
npm run dist    # Создание .exe установщика
```

Установщик появится в папке `dist-electron/`.

> Для сборки Electron-версии потребуется создать отдельный Electron-проект, использующий скомпилированный фронтенд и встроенный SQLite. Инструкции в разделе `electron/README.md` (создаётся при генерации Electron-шаблона).

---

## 5. Где хранится база данных

В веб-версии (Replit) данные хранятся в **PostgreSQL** (облачная БД Replit). Строка подключения — в переменной окружения `DATABASE_URL`.

В будущей Electron-версии данные будут храниться локально в файле **SQLite**:
```
%APPDATA%\СчётчикВремени\database.db
```

---

## 6. Где хранятся резервные копии

В настройках приложения можно задать папку для экспорта (`Настройки → Папка экспорта`).

По умолчанию резервные копии сохраняются рядом с базой данных:
```
%APPDATA%\СчётчикВремени\backups\
```

Автоматические резервные копии создаются раз в сутки. Хранятся последние 30 копий.

---

## 7. Как перенести программу и базу на другой компьютер

**Веб-версия (Replit):** все данные в облачной БД — просто откройте Replit на другом устройстве.

**Electron-версия:**
1. Скопируйте файл `database.db` из `%APPDATA%\СчётчикВремени\` на новый компьютер.
2. Установите программу на новом компьютере.
3. Вставьте `database.db` в папку `%APPDATA%\СчётчикВремени\`.

---

## 8. Как импортировать справочник товаров

1. Откройте раздел **«Справочник товаров»**.
2. Нажмите кнопку **«Импорт CSV»**.
3. Вставьте данные CSV или загрузите файл.

Поддерживаемые колонки в CSV (первая строка — заголовки):
```
Штрихкод,Артикул,Наименование,Категория,Норматив (сек)
4601234567890,SKU-001,Молоко 1л,Молочная продукция,120
```

Если товар с таким штрихкодом уже есть — он будет обновлён. Новые товары добавятся автоматически.

---

## 9. Как выгрузить отчёт

### Из раздела «История»:
1. Откройте раздел **«История»**.
2. Установите нужные фильтры (период, оператор, смена).
3. Нажмите **«Экспорт CSV»**.

### Из раздела «Отчёт»:
1. Откройте раздел **«Отчёт»**.
2. Выберите период.
3. Нажмите **«Экспорт»** для скачивания файла.

Имя файла формируется автоматически, например:
```
Отчет_операции_2026-07-17.csv
```

## Стек технологий

- **Frontend:** React 19, TypeScript, Tailwind CSS, Recharts
- **Backend:** Express 5, Node.js 24
- **База данных:** PostgreSQL + Drizzle ORM (веб), SQLite (Electron)
- **API:** OpenAPI 3.1 + Orval (codegen)
- **Сборка:** pnpm workspaces, Vite, esbuild
